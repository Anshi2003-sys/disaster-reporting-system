About Disaster Reporting System

Helping communities report and monitor disasters efficiently

🌍 What This Website Does

The Disaster Reporting System is a web-based platform that allows citizens to report disasters such as floods, fires, landslides, and earthquakes. The system collects disaster information and displays it on an interactive map so that authorities and communities can respond quickly.

🎯 Project Objective

The main goal of this system is to create a centralized platform where disaster information can be reported, stored, and analyzed. This helps improve disaster response and increases awareness among communities.

Key Features

📝 Disaster Reporting

Users can report disasters easily using an online form.

🗺 Interactive Map

Disaster locations are displayed on a live interactive map.

📊 Analytics Dashboard

Admins can analyze disaster reports using charts and statistics.

System Workflow

User reports disaster → Data stored in database → Admin reviews the report → Disaster displayed on map → Statistics generated for monitoring

Developed as an academic project for disaster monitoring and management.
